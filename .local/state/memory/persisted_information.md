# Persisted Information - AutoClose AI Project

## CONTEXTO FIJO - Archivos de Referencia Principal

### PROJECT_GOLAS.md
**PROJECT_GOLAS.md** es el archivo de referencia principal del proyecto y debe ser consultado siempre para entender la visión, arquitectura y objetivos de AutoClose AI. Este archivo contiene toda la información sobre la estructura del proyecto, principios de desarrollo, integraciones y requisitos.

### design_guidelines.md
**design_guidelines.md** contiene las guías de diseño completas del proyecto AutoClose AI. Este archivo debe ser consultado siempre para mantener la consistencia visual y de UX. Incluye:
- Filosofía de diseño y principios centrales
- Sistema de colores (paleta púrpura institucional)
- Tipografía (Inter, Poppins, JetBrains Mono)
- Sistema de navegación next-gen (Command Palette, Quick Actions, AI Dock)
- Componentes core y microinteracciones
- Responsive behavior y accesibilidad
- Integración visual con Google Workspace
